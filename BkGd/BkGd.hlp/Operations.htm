Wallpaper Changer Operations

This application is composed of two processes:

  • BkGd -- The User Interface Process, accepts the commands below and sends them to BkGDEx
  • BkGdEx -- The Wallpaper Changer Process, it begins running if enabled at login or when BkGd sends a command. If not enabled it immediately exits (stops running).

Tool Bar Commands

Get Current Wallpaper Picture Path

Displays:

  • Picture Root Path
  • Number of wallpaper files found
  • Wallpaper change interval
  • Selection mode, random or linear
  • Previous wallpaper picture full path
  • Current wallpaper picture path
  • Array index of current wallpaper picture

The current picture used as wallpaper is copied to the clip board (useful for pasting the path into a file manaager or text editor). Both the previous and current picture paths may be selected (mouse to the path and press left mouse button). Once selected the path will be highlighted. The a right mouse click will yield a popup menu with a copy command that will put the path into the clip board.

The Get Current Wallpaper Picture Path is executed when the app is started and thus the current picture path is in the clip board.

Next Wallpaper

If the wallpaper changer is disabled, nothing is done (except for a message buried in the middle of the text in the app window). If enabled but not running, the wallpaper changer is started. A message is sent to the wallpaper changer to display the next picture.

Select Root Picture Directory

A Directory Dialog Box is presented to the user. The resulting path is presented to the Wallpaper changer (starting it if necessary).

Select Interval

A dialog box is presented to the user in which to supply the number of minutes to wait before changing the wallpaper. The resulting value is presented to the wallpaper changer (starting it if necessary).

Toggle Selection Mode

There are two modes, random selection of picture to use for the wallpaper and linear selection of the next picture to use for the wallpaper. This command toggles between the two modes and notifies the Wallpaper Changer that the mode has changed.

Print

Will do all the normal things to print the content of the window.

Options

The options dialog allows specifying the Display and Printer Font Size Scale Factor, the printer orientation and the printer margins.

Stop Changing the Wallpaper

Kill the wallpaper changer process (effectively stopping the changing of the wallpaper).

About

Displays the About Box. The about box uses text from the resource file (TestApp.rc) for the output information. There is a standard about box dialog in the Library.

Exit

Exit the application with no reservations. Does not stop the wallpaper changer.

Mouse Operations

The text lines which describe the current and last wallpaper path have a tab after the description and before the actual path. When the mouse cursor is placed anywhere on the path and the left mouse button is pressed the path is "selected" and the font colors are inverted (highlighted). The rule is the selected part of the line is delimited by a tab character on the left and the end of the line.

Selecting a path is only half of the operation. The selected text must be copied into the clip board. This is done in one of the following days:

  • Press Ctrl-C
  • Select Menu/Edit/Copy
  • Double Click the left mouse button instead one click
  • Click the right mouse button and select the Copy command on the Popup Menu
